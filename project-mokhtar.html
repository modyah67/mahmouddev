Mokhtar Ismail Beauty Center Website

Project Overview

A fully professional and responsive website developed for Mokhtar Ismail Center, specialized in 100% natural hair extensions using advanced Ring Layers technique. The goal was to create a premium online presence that reflects trust, elegance, and high service quality.

Main Features

  • Modern fully responsive design (Mobile / Tablet / Desktop)
  • Hero slider with direct WhatsApp booking button
  • TikTok video integration for real client transformations
  • About page with service details & Google Maps location
  • Products page with colors, lengths, weights & prices
  • Instant WhatsApp inquiry per product
  • Optimized loading speed & SEO basics

Technologies Used

HTML5 • CSS3 • Bootstrap 5 • JavaScript

Full WhatsApp integration allowing instant booking and inquiries directly from the website.
