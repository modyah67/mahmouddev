AI Disease Diagnosis System

I developed a smart disease diagnosis system based on modern technologies. The frontend was built using Flutter to create a simple and responsive interface, while Python was used for data processing and implementing Artificial Intelligence.

The main goal of the system is to help users get a quick preliminary health evaluation, enabling them to take informed initial steps regarding their condition.

How The System Works

Health Data Input

Users can enter basic personal health information such as weight, age, Body Mass Index (BMI), blood pressure level, and blood sugar level.

Analysis & Prediction

The system processes this data using intelligent algorithms to determine the most probable disease and assess the severity level, providing a clear initial evaluation.

Symptom-Based Diagnosis

The system allows flexible symptom input either through categorized organized lists or manual free-text entry.

After submission, symptoms are analyzed carefully to provide a more detailed and accurate potential diagnosis, focusing on both speed and precision.

Additional Features

  • Nearest hospital detection using GPS location services.
  • Simple, fast, and responsive user interface.
  • Fully compatible with mobile phones and tablets.
  • Scalable architecture ready for integration with medical databases or external systems.
